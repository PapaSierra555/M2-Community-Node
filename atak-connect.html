ATAK Setup

M2 Community Node — Tactical Map Enrollment

ATAK gives you a live shared map with team positions, markers, and encrypted messaging — all offline, over the CommunityNode WiFi.

Follow the steps below in order. The download link is at the bottom. Read this page first so enrollment goes smoothly.

Setup Steps

  1. Install the ATAK app on your device. ATAK-CIV is Android only. iPhone users install iTAK from the App Store.
    ATAK-CIV Android
    iTAK iPhone
  2. Install a voice app for push-to-talk team comms over WiFi. Android uses Mumla, iPhone uses Mumble — they connect to the same server.
    Mumla Android
    Mumble iPhone

    After connecting: open your voice app → add server → 192.168.8.20, port 64738. No password needed.

  3. Open ATAK and grant all requested permissions — location, battery optimization exemption, and storage. ATAK needs these to track your position and store map data.
    Do not connect ATAK to any server manually. The next step handles enrollment automatically.
  4. Import the connection package using the button below. Tap Open in ATAK first — it opens ATAK and imports automatically. If that button does not work on your device, use the zip download and import manually.

    Manual import: Download the zip → open ATAK → hamburger menu (≡) → Import Manager → Local SD → navigate to Downloads → select the zip → OK.

    Your icon appears on the shared map. You’re in. If ATAK prompts for credentials, ask your operator.
  5. Set your callsign. Settings → My Preferences → Callsign. Use your name or the callsign assigned by the operator. This is how you appear on the team map.
  6. Set your reporting mode. Settings → My Preferences → Reporting Preferences → Reporting Strategy → set to Consistent. This ensures your position is sent on a fixed interval. The default Dynamic mode only reports when you move — a stationary device will not appear on the map.
  7. Verify your connection. Your icon should be visible on the map within 30 seconds. Send a test message in GeoChat → General to confirm you’re on the team net.

    Not showing on map? Go to Settings → Network Preferences → TAK Servers → tap your server → toggle Enabled off then back on. Wait 30 seconds. This forces a fresh connection and SA report.

Step 4 — Connect to M2 Node

Android (ATAK): Tap Open in ATAK — opens ATAK and imports automatically.

Open in ATAK Android only — opens ATAK and connects automatically

Fallback / iOS: Download the zip, then import manually in ATAK or iTAK.

Download atak-connect.zip M2 Community Node — connection package

Cert warning? Tap Advanced → Proceed to 192.168.8.20. Expected on a private network with a local certificate.

Manual Connection (If Package Fails)

TAK Server Settings

Server 192.168.8.20
Port 8089
Protocol SSL
Credentials Ask your operator

ATAK → Settings → Network → TAK Servers → Add. Enable Use Authentication and Enroll for Client Certificate. Uncheck Use default SSL/TLS Certificates. Truststore password from your operator.

Not on the WiFi?

Browser map at tak.yourdomain.com — no app needed, full shared map, email OTP login.
