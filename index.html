Tactical Map — ATAK Setup

ATAK gives you live positions, markers, and team chat on a shared map. These steps are for phones on the Community Node WiFi. Not here? Use the web map at tak.yourdomain.com — no app needed.

  1. Install ATAK-CIV from the Play Store (Android only). Tactical map with live team positions, markers, and messaging.
    ATAK-CIV
  2. Install Meshtastic plugin — bridges ATAK over LoRa mesh radio so your position and messages work without WiFi or cell service. Download the APK from GitHub (not on Play Store).
    Meshtastic (GitHub)
  3. Install Mumla — push-to-talk voice chat with your team, like a radio but over WiFi.
    Mumla
  4. Open ATAK — grant location and battery optimization permissions when prompted.
  5. Ask the operator to connect your device to the server. They will configure your connection and callsign.
  6. Voice: Open Mumla, connect to 192.168.8.20 port 64738. No password needed.
