# M2 Community Node — QR Codes

Two separate QR workflows exist in this project. This directory holds the output of both.

---

## What's in This Directory

### Script-generated SVGs (`svg/*.svg`)

These are generated by `scripts/generate_qr.py` and embedded in the PDF documents and kiosk web pages. They encode URLs and strings that come from your `instance.conf`.

| File | Encodes | Used In |
|---|---|---|
| `qr-element-lan.svg` | `http://192.168.8.10:8080` (Element on local WiFi) | Kiosk modal |
| `qr-atak-android.svg` | Google Play — ATAK CIV | Kiosk modal |
| `qr-itak-ios.svg` | App Store — iTAK | Kiosk modal |
| `qr-meshtastic.svg` | GitHub — Meshtastic ATAK plugin | Kiosk modal |
| `qr-mumla.svg` | Google Play — Mumla (Mumble client) | Kiosk modal |

> **The files above encode public app store URLs or the reference IP (192.168.8.10) and contain no instance-specific data. They can be used as-is.**

The remaining QR codes (WiFi, clearnet URLs, Tor .onion addresses) are **not committed** because they contain instance-specific values. Regenerate them for your build:

```bash
pip install qrcode
python scripts/generate_qr.py
```

This reads your `instance.conf` and writes all QR SVGs to `svg/`. Run it any time your IPs, domains, or WiFi password change.

---

### Vinyl-cut reference set (`svg/WebQrCodes/`)

This subdirectory holds the style configuration used to generate vinyl-cut QR codes in [mini-qr](https://github.com/lyqht/mini-qr). These codes go on the physical rack panel.

| File | Purpose |
|---|---|
| `qr-code-config.json` | mini-qr style preset — dot type, colors, size, frame label |

> **No SVG files are committed here.** You must generate your own vinyl-cut QR codes with your actual URLs. Import `qr-code-config.json` into mini-qr to reproduce the reference style.

The four panel QR codes to generate in mini-qr:

| Panel Label | Encodes |
|---|---|
| WIFI | `WIFI:T:WPA;S:CommunityNode;P:YOUR_WIFI_PASSWORD;;` |
| COMMUNITY | `https://yourdomain.com` |
| ELEMENT WEB | `https://communitynode.yourdomain.com` |
| ATAK CONNECT | `https://atakenroll.yourdomain.com:8447` |

See `docs/QR_CODES.md` for the full workflow including vinyl cutting notes.
